# Lelani School Assets

This directory contains the school's branding assets.

## Required Files

### 1. logo.png
- **Purpose:** School logo displayed on login page and reports
- **Recommended size:** 500x500px (square)
- **Format:** PNG with transparent background
- **Usage:** Login page header, PDF reports

### 2. favicon.png
- **Purpose:** Browser tab icon
- **Recommended size:** 32x32px or 64x64px
- **Format:** PNG
- **Usage:** Browser favicon

## How to Add Your Logo

1. Save your Lelani School logo as `logo.png` in this directory
2. Save your favicon as `favicon.png` in this directory
3. Ensure the logo has a transparent background for best results
4. The logo should be high quality (at least 500x500px)

## Temporary Placeholder

If you don't have the logo files yet, you can:
1. Create simple placeholder images
2. Use a logo generator online
3. Or the system will show a broken image icon until you add them

## Color Reference

Make sure your logo uses the official Lelani School colors:
- **Primary Red:** #D32F2F
- **Dark Grey:** #333333
- **White:** #FFFFFF
